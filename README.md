# 🍃 Clean-Green

**Clean-Green** is a modern MERN-stack web application designed for citizens to report environmental and waste management issues directly to the authorities. By leveraging interactive maps and real-time reporting, it bridges the gap between the public and municipal departments to ensure a cleaner and greener environment.

---

## 🚀 Key Features

- **📍 Interactive Map Reporting**: Pinpoint the exact location of waste or environmental hazards using Leaflet maps.
- **📷 Evidence Upload**: Upload photos of the site to provide visual proof for faster processing.
- **🏷️ Smart Categorization**: Reports are classified into Municipal, Bio-medical, Electronic, and Plastic/Toxic waste.
- **📢 Official Gmail Dispatch**: Automatically generates and sends professional grievance petitions to the relevant authorities via Gmail interface/Nodemailer.
- **🔐 Secure Authentication**: User-specific dashboard with JWT-based login and registration.
- **📊 Live Tracking Feed**: View a public feed of reported issues with their current status (e.g., "Cleaned" or "Awaiting Repair").
- **📱 Responsive UI**: Beautifully designed with CSS and Framer Motion for a premium user experience across all devices.

---

## 🛠️ Tech Stack

### Frontend
- **React.js** (Vite)
- **Leaflet & React-Leaflet** (Interactive Maps)
- **Framer Motion** (Micro-animations)
- **Lucide-React** (Modern Iconography)
- **Axios** (API Requests)
- **React Router** (Navigation)

### Backend
- **Node.js & Express.js**
- **MongoDB & Mongoose** (Database)
- **JWT (JSON Web Tokens)** (Authentication)
- **Multer** (File/Image Upload)
- **Nodemailer** (Email Automation)
